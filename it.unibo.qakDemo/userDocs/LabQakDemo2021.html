The project it.unibo.qakDemo

CallRestWithApacheHTTP.kt

A Kotlin object that provides the following operations:
  • doPath: calls the API doPath of the service localhost:8081 and returns the payload of the response (ApplMessage) as answer
  • doMove: calls the API moverest of the service localhost:8081 and returns the payload of the response (ApplMessage) as answer

SonarObserver.kt

An actor of type ActorBasicKotlin.kt (2021) that handles messages (of type ApplMessage.kt (2021)) related to the sonar sent by the WEnv and uses the master registered in pathexecutil.kt to generate an event sonar : distance(D) that can be perceived by all the components of the system, including the master.

pathexecutil.kt

A Kotlin object that provides the following operations:
  • memoCurPath
  • nextMove
  • doMove: send a command to move the robot by using CallRestWithApacheHTTP.k and send to the caller a dispatch (moveok|movefail) related to the answer
  • createSonarObserver: creates an IssWsHttpKotlinSupport.kt (2021) on the websocket 8091 and registers to that support the SonarObserver.kt.

boundarywalker.qak

Uses doMove of pathexecutil.kt, builds a map of the room and handles the alarms. It activates also the SonarObserver.kt that will show the data emitted by WEnv.
